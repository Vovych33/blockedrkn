# BlockedRKN

**BlockedRKN** — это моя модификация оригинального проекта `zapret-discord-youtube`, созданная для личного использования и удобства русскоязычных пользователей.

### Что было сделано:
* **Перевод на русский язык**
* **Улучшение интерфейса**
* **Удобный установщик**

## 📌 О авторстве

> **Важно:** Я не являюсь автором оригинальной утилиты `zapret-discord-youtube`. Вся техническая база и логика работы полностью принадлежат разработчику [Flowseal] (https://github.com/Flowseal/zapret-discord-youtube).

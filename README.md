# BlockedRKN

Удобная графическая оболочка и локализация для популярной утилиты обхода блокировок **zapret**.

## 📌 О авторстве

> **Важно:** Я не являюсь автором оригинального утилиты `zapret`. Вся техническая база и логика работы полностью принадлежат разработчику [Flowseal] (https://github.com/Flowseal/zapret-discord-youtube). 

**BlockedRKN** — это моя модификация оригинального проекта, созданная для личного использования и удобства русскоязычных пользователей.
